Sākums

Spiežot pogas "Iepriekšējais" un "Nākamais" kartiņas parādīsies pa mēnešiem un tiks paskaidrots, ko es apguvu šajā laikā. Kā arī Jums ir iespēja atrast konkrētu mēnesi.

Septembris

Šajā laikā vēl netika apskatīta jauna tēma.

Oktobris

Šajā laikā vēl netika apskatīta jauna tēma. Oktobra beigās sākās praktiskā pieredze uzņēmumā.

Novembris

Šajā mēnesī guvu praktisku pieredzi uzņēmumā. Tika izmantots HTML un JavaScript. Es apguvu HTML un JavaScript pamatus praktiskajā pieredzē uzņēmumā.

Decembris

Prakse beidzās decembra sākumā. Programmēšanas stundās apskatījām TypeScript funkcijas un pildījām uzdevumus.

Janvāris

Janvārī turpinājām apskatīt TypeScript funkcijas.

Februāris

Februārī jau apskatījām HTML, JavaScript un CSS. Man jau pamati tika apgūti priekš HTML un JavaScript praktiskajā pieredzē uzņēmumā, taču ieguvu pamatus CSS. Arī tika izveidota prezentācija pēc izvēlētas tēmas un tika izveidots pirmais projekts "Kalkulators". (Projektu/-s būs iespējams apskatīt!)

Marts

Martā jau tapa nākamie 2 projekti - "Datu aizpildes veidlapa" un "Objektu kārtis". (Projektu/-s būs iespējams apskatīt!)

Aprīlis

Aprīlī tika papildināts "Objektu kārtis" ar papildus funkcijām un tika veikts nākamais projekts - "Mērķlapa". (Projektu/-s būs iespējams apskatīt!)

Maijs

Maijā tika iesākts projekts - "Gala darbs", kurā ir iespējams novērot visu apgūto laika gaitā. (Projektu/-s būs iespējams apskatīt!)
